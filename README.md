# **React Vite Beginner Tasks**

This is a **React project built with Vite** that demonstrates **10 beginner React tasks** using functional components, props, state management, list rendering, simple form handling, and dummy API fetching. The project is minimal and beginner-friendly, perfect for learning React basics in a modern setup.

---

## **Features**

* `WelcomeMessage` – Displays a welcome heading
* `Header` & `Footer` – Render multiple components together
* `Greeting` – Shows personalized text using props
* `Counter` – Button click counter using `useState`
* `ToggleText` – Shows/hides a paragraph using state
* `LiveInput` – Updates text live as user types
* `FruitList` – Displays an array of fruits using `.map()`
* `AddFruit` – Input + button to add new items to a list
* `LoginForm` – Simple form validation for empty fields
* `FetchPosts` – Displays first 10 post titles (clean dummy data)

All components include **minimal CSS styling** for a clean, visually pleasant look.

---

## **Tech Stack**

* **React** - Frontend library for building UI components
* **JavaScript (ES6+)** - Logic and JSX rendering
* **CSS** - Styling the components
* **Vite** - Fast dev server and build tool

## **Getting Started**

Follow these steps to run the project locally:

1. **Clone the repository:**

```bash
git clone https://github.com/your-username/react-vite-beginner-tasks.git
```

2. **Navigate to the project folder:**

```bash
cd react-vite-beginner-tasks
```

3. **Install dependencies:**

```bash
npm install
```

4. **Run the development server:**

```bash
npm run dev
```

Open your browser at [http://localhost:5173](http://localhost:5173) to see it live.

---

## **Customization**

* You can edit any component in the `src/components` folder. For example, `WelcomeMessage.jsx`:

```javascript
function WelcomeMessage() {
  return <h1>Your Custom Message Here!</h1>;
}
```

* Modify each component’s CSS to change:

  * Background color
  * Font styles
  * Text color
  * Padding/margin

* **`npm run build`**: Builds the app for production into `dist/`.
